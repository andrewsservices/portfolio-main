Job Skills App

Front-End Project (React.js)

  • HTML5, CSS3, JSX, and Reactjs.

  • Utilized MUI React Library for styling.

  • Built Reusable Form Component for adding multiple jobs and skills.

  • Utilized UseState Hook to update State throughout app.

  • Onclick Event Listener increases skill count for each skill.

  • Sort button activates custom JavaScript function which re orders skills based on skill count.
